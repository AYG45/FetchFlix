# FetchFlix - Anime Streaming Platform

A simple anime streaming site built with React and TypeScript.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Features

- Browse popular and trending anime
- Search by title
- Multi-season support
- 5 streaming sources
- Watch history

## How It Works

1. Fetches anime data from Jikan API (MyAnimeList)
2. Gets TMDB ID for streaming
3. Provides 5 embed players
4. Switch sources if one doesn't work

## Tech Stack

- React + TypeScript + Vite
- Jikan API (anime data)
- Embed players (streaming)

## Note

Works best with popular anime that have TMDB IDs (Demon Slayer, Attack on Titan, etc.)
